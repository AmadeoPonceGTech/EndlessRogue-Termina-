# 🎮 Endless Rogue

The player chooses **4 heroes among 8 available** and upgrades them through gameplay to survive increasingly difficult waves of enemies.

# 🔄 Gameplay

## Player Objectives

* Survive as long as possible in the **infinite dungeon**
* Kill monsters to gather **upgrade materials**
* Upgrade characters to **unlock abilities and improve stats**

---

## Gameplay Loop

1. Start a run
2. Fight monsters (boss wave each 10 levels)
3. Collect resources and artefacts
4. Upgrade and manage characters
5. Start a new run and go further

---

## Controls

| Input          | Action         |
| -------------- | -------------- |
| **Left Click** | Select actions |

---

# 🧙 Characters

The player chooses **4 heroes from 8 available characters** : Alex, Penelope, Diane, Emilie, Claire, Marcus, Brutus and Edward.

---

# 👹 Enemies

There are **16 differents enemies**. Each enemies can drop artefacts, who can be assigned to a character to boost his power.

# 📖 How to install/use

To launch EndlessRogue, you have to install xmake and add to path. You will also need Vulkan SDK.

When you launch the Termina Engine, you have to create an entity with the Component `Game`.
Before this, you can just start the game, and enjoy !
